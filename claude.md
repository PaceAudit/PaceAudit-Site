# CLAUDE.md - PaceAudit Project Instructions

## Project Intent
PaceAudit is a high-end SaaS for construction auditing. The brand must feel trustworthy, modern, and expensive.

## Design Skill: UI/UX PRO MAX
- Follow the guidelines in .cursorrules for all UI generation.
- Use 21st.dev as the primary reference for React/HTML components.
- Maintain a consistent "Dark Mode" aesthetic using Slate-900 for backgrounds and Indigo-500 for primary accents.

## Build & Test Commands
- **Install:** `npm install`
- **Dev Server:** `npm run dev` (using Vite)
- **Deploy:** `netlify deploy --prod`

## Coding Standards
- Standard: Functional, clean, and modular code.
- CSS: Tailwind CSS (Configured in tailwind.config.js).
- Auth: Supabase Auth helpers.
- Images: Use Nano Banana 2 generated prompts for placeholders.

## Formatting
- Use 2-space indentation.
- Always include helpful comments in JavaScript for logic flow.
